# TechShop SQL Project

## Overview
A beginner SQL project built from scratch using MS SQL Server.
Designed a 4-table relational database for a tech shop and 
wrote 5 business queries.

## Tables
- Customers
- Products
- Orders
- Order_Items

## Business Queries
1. Which customer spent the most money?
2. Which product sold the most units?
3. Total revenue per category
4. Orders with Express shipping
5. Total revenue per customer

## Tools Used
- MS SQL Server
- SQL Server Management Studio (SSMS)

## Skills Demonstrated
- Database design from scratch
- CREATE TABLE with constraints
- PRIMARY KEY & FOREIGN KEY
- INSERT INTO statements
- JOIN across 3 tables
- GROUP BY, SUM, ORDER BY
- WHERE filtering
